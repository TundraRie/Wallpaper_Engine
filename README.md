# Wallpaper_Engine
Wallpaper Engine - анимированные обои!

# Об этом ПО
Wallpaper Engine позволяет устанавливать живые обои на рабочий стол Windows. Поддерживаются различные типы анимированных обоев, включая 3D- и 2D-анимации, веб-сайты, видео и даже определённые приложения. Вы можете создавать собственные обои и использовать их локально или делиться с другими напрямую (не через сторонние сервисы). Все обои добавляются вручную из ваших файлов или создаются во встроенном редакторе.

# Возможности
• Оживите свой рабочий стол благодаря анимированной графике, видео, приложениям и веб-сайтам
• Используйте анимированные скринсейверы, когда вы отошли от компьютера
• Персонализируйте анимированные обои, используя любимые цвета
• Используйте интерактивные обои, с которыми можно взаимодействовать при помощи мыши
• Поддерживается множество вариантов соотношения сторон и собственных разрешений, например 16:9, 21:9, 16:10, 4:3
• Поддержка систем с несколькими мониторами
• Анимация обоев будет приостановлена во время игр для экономии производительности
• Создавайте собственные анимированные обои в приложении Wallpaper Engine Editor
• Анимируйте новые живые обои из обыкновенных файлов изображений или импортируйте HTML или файлы видео для создания обоев
• Все обои добавляются через локальные файлы (поддерживаются форматы: mp4, WebM, avi, m4v, mov, wmv)
• Поддержка Razer Chroma и Corsair iCUE

# Производительность
Приложение Wallpaper Engine нацелено на развлечение пользователей, при этом используя как можно меньше ресурсов системы. Живые обои можно автоматически приостановить или отключить полностью при использовании другого приложения или игре в режиме полного экрана (включая оконный режим без рамок), чтобы не отвлекаться во время игры или работы. Множество параметров, позволяющих лавировать между качеством и производительностью, позволяют идеально интегрировать Wallpaper Engine в вашу систему.

Как показывает опыт, 3D, 2D и видео обои задействуют меньше ресурсов системы, в то время как веб-сайты и приложения в качестве обоев требуют гораздо большей производительности. Рекомендуется использовать дискретную видеокарту, но это не обязательно.

# Творчество и библиотека
Дайте волю фантазии и используйте Wallpaper Engine Editor для создания собственных анимированных обоев из изображений, видео, веб-сайтов и приложений. Большой выбор предустановок и эффектов позволят вам анимировать собственные изображения. Все обои хранятся локально и добавляются вручную через приложение.

# Системные требования

# Минимальные:

• ОС: Windows 10, Windows 11
• Процессор: 1.66 GHz Intel i5 или аналог
• Оперативная память: 1024 MB ОЗУ
• Видеокарта: HD Graphics 4000 или выше
• DirectX: версии 11
• Место на диске: 512 MB
• Дополнительно: Для версий Windows N требуется Media Feature Pack от Microsoft

# Рекомендованные:
• ОС: Windows 10, Windows 11
• Процессор: 2.0 GHz Intel i7 или аналог
• Оперативная память: 2048 MB ОЗУ
• Видеокарта: NVIDIA GeForce GTX 660, AMD HD7870, 2 GB VRAM или выше
• DirectX: версии 11
• Место на диске: 1024 MB
